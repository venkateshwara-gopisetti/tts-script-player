Quick Reference
🔌 Audio Routing Windows Volume Mixer → set this browser's output to CABLE Input (VB-Audio) so TTS plays as mic input into your recording app.
⏱ Timeline Mode Enter M:SS or M:SS.s on any card (e.g. 0:05, 1:30.5), then hit ▶ Run Timeline. Lines fire at their timestamps automatically, like SRT subtitles driving audio.
✦ Card Controls Edit the index number and press Enter to jump a card to any position. Drag the handle to reorder. Cards with timestamps get an orange left border.
